1. Basic RP Terminology

> ​IC (In Character): Game eka athule oya inna character eka widiyata hasirenna ona. > > ​OOC (Out of Character): Game eken pita nawa katha kirima. (Meeta "T" key eka use karana eka hari Discord eka hari use wenawa).
> Metagaming: Game eken pita (Discord, YouTube live streams) dena gaththa thorathuru game eka athule character ekata therenawa wage use kirima thahanam.
> > ​Powergaming: Character ekakata karanna bari dewal (real life eke karanna bari dewal) game eke kireema. Udaharanayak widiyata, accident ekak wela loku damage ekak unath kisi deyak une na wage aye duwana eka. >


2. Death & Combat Rules

> ​RDM (Random Deathmatch): Kisiama hethuwak nathuwa thawa kenekwa marana eka thahanam. Maranna kalin hari RP reason ekak thiyenna ona.
> ​VDM (Vehicle Deathmatch): Wahana walin happala minissu marana eka sampurnayenma thahanam.
> ​NLR (New Life Rule): Oya game eke marunama, oya eeta kalin una dewal mathaka na kiyala hithanna ona. Marunu thanata winadi 15-30k yanakan aye yanna ba.
> ​Combat Logging: Fight ekak hari police chase ekak hari yaddi game eken quit wena eka thahanam.


3. Roleplay Etiquette

> Value Your Life: Thaman ge jiwitheta watinakemak denna ona. Udaharanayak widiyata, kawru hari gun ekak allala thiyeddi baya nathuwa "gahapan" kiyala kiyanna ba. Real life eke wage baya wenna ona.
> ​Fail RP: RP ekak asse thamange character eken pita hasirima (Udaharanayak: Police karayek ekka katha karaddi "Meka game ekakne" kiyala kiyana eka).
> ​Fear RP: Thamanata wada balawath ayek (gun ekak athi ayek) idiriyadi baya wiya yuthui. >


4. Vehicle & Property Rules

> Safe Zones: Hospital, Police Station, and Spawn point wage thanawal "Safe Zones" widiyata salakanne. Ethana kauruwath maranna hari robbery karanna hari ba.
> ​Car Jacking: Hehetuwak nathuwa hamathanama thiyena wahana horakam kirima thahanam.`
